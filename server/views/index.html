Creating a collaborative and creative virtual environment on the Web

Collavrate is a VR application built entirely on the Web. It allows users to collaborate and create 3D models while being in a virtual environment. By leveraging the Myo armband, Collavrate enables users to draw, model, and control movement using gestures.

Collaborative Whiteboard

3D Modelling

Create 3D models by extruding the 2D shapes that you draw, and then doing simple mathematical operations like subtraction, intersection, and union.

This project would not have been possible without:

  • Three.js
  • Myo.js for Myo Javascript Bindings
  • csg.js for combining 3D solids
  • The MozVR project
